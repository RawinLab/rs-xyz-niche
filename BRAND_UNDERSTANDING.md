## 3. Logo Assets (ใน PDF มี 3 options + 1 submark)

### OPT 01 — Bold geometric, chamfered
- Wordmark "NICHE" — sans-serif หนา ตัวเหลี่ยม ตัด corners แบบ stencil/Eurostile
- N, I, C, H, E ทุกตัวมี chamfered cuts
- Feel: techy / architectural / utilitarian / industrial

### OPT 02 — Bold rounded humanist
- Wordmark "NICHE" — rounded humanist sans หนา
- **N เอียง** (tilted) — เป็น mnemonic ที่จำง่าย
- **C ไม่สมมาตร** — ปลายตัด/เปิดแบบ asymmetric
- Feel: confident / editorial / magazine-cover / approachable

### OPT 03 — Thin decorative display
- Wordmark "NICHE" — เส้นบาง, art-deco/fashion energy
- **i แทนด้วยวงกลม/จุด** — concept-rich (วง = niche/focal point)
- Feel: boutique / luxurious / fashion / architectural-thin

### Submark — "N."
- Letter N รูปทรงบ้าน (house-shape) + period (.)
- ใช้ได้ทั้งแบบขาวบนดำ/ดำบนขาว
- Function: app icon / favicon / IG avatar / embroidery / embossing
- Concept: period punctuates "carved niche" + reads as declarative "NICHE."

## 5. Typography (ใน PDF — ที่ใช้จริงในการออกแบบ)

ไม่มี type spec sheet ระบุชื่อ font ตรง ๆ แต่จาก mockups สังเกตได้ว่า:

- **Display/headline** — geometric sans หนา, รูปร่างใกล้เคียง Aeonik / Söhne / GT Walsheim (เช่น "Find / your / niche" บนบิลบอร์ด)
- **Body / UI** — geometric sans regular, ใกล้เคียง Inter / TWK Everett
- **Logo type** — custom drawn ทั้ง 3 options (ไม่ได้ใช้ font สำเร็จ)
- **No Thai type appears** ใน mockups (ทุกอย่างเป็นอังกฤษ)

> หมายเหตุ: spec ที่ผมแนะนำในเอกสาร research (GT Sectra + Bai Jamjuree) เป็นข้อเสนอใหม่ ไม่ใช่ของเดิมที่ designer ใส่มา

## 8. Photography Direction (สังเกตจาก mockups)

- เน้นภาพ **มือทำงาน** — annotating sketches, ใช้ laptop, อ่านแมกกาซีน, จับ chair, sketching
- มี physical artifact ในเฟรม: sticky notes, sketchbooks, color swatches, magazine spread, blueprints
- Lighting: natural / window-lit · ไม่ใช่ studio-polished
- Color tone: warm-natural, ไม่ฟิลเตอร์มาก — รักษา skin tone จริง
- Subjects: candid (ไม่มีใครมองกล้อง) · workspace cluttered-but-intentional
- Crop: ทั้ง close-detail (มือ + กระดาษ) และ environmental (ห้อง + คน)

## 9. สิ่งที่ **ยัง ไม่มี** ใน PDF (gap analysis)

ตรงนี้สำคัญสำหรับการเขียน brand book ฉบับเต็ม — สิ่งที่ designer **ไม่ได้** ใส่มา:

- ❌ **Logo construction grid** (geometry, proportion, clear-space, min-size)
- ❌ **Logo do / don't** rules
- ❌ **Pillar / Program lock-up system** (NICHE × Pillar × Program ยังไม่มีโครง)
- ❌ **Type spec sheet** (font name, scale, weights, line-height, tracking)
- ❌ **Color usage rules** (which color for what — ไม่มี do/don't, ไม่มี WCAG audit)
- ❌ **Dark-mode token alternates**
- ❌ **Photography direction guide** (mood board, do/don't, color grade recipe)
- ❌ **Motion / animation system**
- ❌ **Voice & tone framework** (มีแค่ตัวอย่าง copy ไม่มี principles + do/don't)
- ❌ **Iconography system**
- ❌ **Course card / certificate / badge templates**
- ❌ **Faculty / staff / alumni templates**
- ❌ **Event & wayfinding kit**
- ❌ **Accessibility / WCAG documentation**
- ❌ **AI content policy**
- ❌ **Partnership / co-branding rules**
- ❌ **File naming + asset library structure**
- ❌ **Governance / version log**

## 11. Working Hypotheses (ของผม — อาจผิดได้)

- **NICHE น่าจะตั้งใจเปิด cohort พฤศจิกายน 2026** (จาก poster "11–30 November 2026")
- **DESIGN AND CRAFT COURSE** อาจเป็น flagship/แรก ก่อนเปิด program อื่น
- **Designer สร้าง 3 logo options ไว้ให้ stakeholder เลือก** — ไม่ได้ตั้งใจให้ใช้ทั้ง 3
- **Brand identity ยังเป็น exploration phase** — ไม่ใช่ final brand book — ขาด construction/spec อีกเยอะ
- **Studio น่าจะอยู่ไทย** (จาก context ของผู้ใช้) แต่ภาษาในมาเฉพาะอังกฤษ → กลุ่มเป้าหมายคือคนไทยที่ aspirational/อ่านอังกฤษ หรือ international?
