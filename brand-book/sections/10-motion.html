Section 10

Motion & Animation System

Five tokens. Three principles. Without this, every video editor improvises and every website feels like a different brand.

Five Motion Tokens

Authority: IBM Carbon Motion. Adapted for NICHE.

Five durations cover everything NICHE will need. Reference the CSS variable, never the millisecond value directly. When the system recalibrates, one change propagates everywhere.

Token Duration Bar Use
--motion-micro 80ms
Icon state changes, checkbox toggles, hover fills. Imperceptible transitions that prevent jarring state snaps.
--motion-quick 150ms
Button press, tooltip appear, tag hover, card background cross-fade. The primary interaction token.
--motion-base 280ms
Card entrance, drawer open, modal appear, accordion expand. System transitions that communicate structure.
--motion-slow 500ms
Page transitions, hero text reveals (per word). These transitions carry narrative weight — they should feel considered.
--motion-logo 900ms
Logo reveal — staged three-part sequence. See logo reveal spec below. Use only at page load / splash.
Easing Curves

Three curves. Every animation uses one of these.

Linear easing is never approved. Everything enters, exits, or adjusts in place — each requires its own physics.

Entrance cubic-bezier(0, 0, 0.3, 1)

Fast start, slow settle. For all elements entering the viewport or appearing on screen. Feels decisive, not tentative.

Exit cubic-bezier(0.4, 0.14, 1, 1)

Slow start, fast end. For elements leaving the screen. Pulls away with intention — doesn't linger.

Standard cubic-bezier(0.2, 0, 0.38, 0.9)

For in-place state changes: color swaps, hover fills, tab selections. Balanced — not theatrical.

Three Motion Principles

Motion earns its place. It doesn't decorate.

1
Type-first reveal

Display text enters word-by-word — not letter-by-letter. Letter-by-letter is the startup cliché. 40ms stagger per word. Fraunces deserves to be read, not animated.

2
Red activates on intent

#fe1d25 appears in motion only on hover/active states. It punctuates; it never auto-decorates on scroll. Red that fires without user intent trains users to ignore it.

3
Program cards cross-fade

On hover: background cross-fades #f2f1f0 → #252026 at quick (150ms). Category tag scales 1.0 → 1.05 simultaneously. The card reacts to presence — not to arrival.

Logo Reveal Specification

900ms. Three stages. One breath.

The tilted N in OPT 02 creates a natural motion behavior. The full logo reveal is a staged sequence — mark first, wordmark second, tagline third. Use only at page load or brand splash moments.

N. submark 0ms → 400ms
Wordmark 350ms → 700ms
Tagline 650ms → 900ms
Stage 1 — Submark N. enters from below at translateY(20px) opacity 0 → 1. Duration: 400ms. Easing: entrance. The N's tilt animates from 0° to final angle.
Stage 2 — Wordmark Full NICHE wordmark cross-fades in starting 350ms. Overlap with Stage 1 creates continuity. Duration: 350ms. Easing: entrance.
Stage 3 — Tagline "Find your niche." fades in word by word, 40ms stagger. Starts 650ms. Duration: 250ms. Easing: standard.
Total duration 900ms from first frame to complete. Do not extend — momentum is the point.
Reduced motion All durations → 0ms. Mark, wordmark, and tagline appear simultaneously. No animation. This is required — not optional.

Prohibited Motion Effects

These effects are banned. Not discouraged — banned.

Every prohibited effect below was either a cliché in 2022 or creates accessibility harm. Neither is acceptable at NICHE.

Letter-by-letter reveal

The startup-pitch cliché. NICHE uses word-by-word only. Letter-by-letter feels performative, delays legibility, and fails in reduced-motion contexts.

Parallax scroll

Scroll-linked position parallax causes motion sickness (vestibular disorder). Violates WCAG 2.3.3. Not allowed anywhere in the NICHE web system.

Auto-playing video with motion

Video that plays automatically with significant motion must include a pause control. Autoplay with motion without pause = WCAG 2.2.2 failure.

Bounce easing

bounce-in, bounce-out — toy register. NICHE is editorial. Use the three approved curves. If it needs to bounce, the design is wrong.

Infinite loop animations

Spinning logos, pulsing badges, looping backgrounds. They fight for attention without earning it. Motion at NICHE is triggered by user intent, not ambient.

Flash / strobe

Three flashes per second or more causes seizures. WCAG 2.3.1. No flashing at any frequency in NICHE digital materials. No exceptions.

CSS Implementation

Copy this. Reference the tokens. Don't rewrite it.

/* Motion tokens — already defined in tokens.css */
--motion-micro:   80ms;
--motion-quick:   150ms;
--motion-base:    280ms;
--motion-slow:    500ms;
--motion-logo:    900ms;

--ease-entrance:  cubic-bezier(0, 0, 0.3, 1);
--ease-exit:      cubic-bezier(0.4, 0.14, 1, 1);
--ease-standard:  cubic-bezier(0.2, 0, 0.38, 0.9);

/* Example: program card */
.program-card {
  transition:
    background  var(--motion-quick) var(--ease-standard),
    color       var(--motion-quick) var(--ease-standard);
}

/* Example: word-by-word reveal */
.word-inner {
  display: inline-block;
  transform: translateY(110%);
  transition: transform var(--motion-slow) var(--ease-entrance);
}
/* Apply stagger in JS: delay = wordIndex * 40ms */

/* Reduced motion — REQUIRED */
@media (prefers-reduced-motion: reduce) {
  :root {
    --motion-micro:  0ms;
    --motion-quick:  0ms;
    --motion-base:   0ms;
    --motion-slow:   0ms;
    --motion-logo:   0ms;
  }
}
