Fraunces handles the moments that demand presence — hero headlines, pull quotes, section titles. Bai Jamjuree handles everything else: body copy, UI labels, data tables, captions, Thai and Latin simultaneously. They don't compete. They take turns.

Brand Book — Section 03

Every violation listed
here has happened.

The tilted N is not a styling choice — it is a mnemonic. It makes NICHE recognizable before the full wordmark registers. Rotating the logo cancels this. Stretching it cancels this. Recreating it in Helvetica cancels this.

Usage Rules

Don't improvise the type system.

Do
Use Fraunces only for display: hero headlines, section intros, pull quotes, cover. At 24px and above.
Don't
Use Fraunces for body copy, UI labels, navigation, or captions. Below 24px it becomes illegible at body density.
Do
Reference CSS custom property tokens in all code: --font-display, --font-body, --type-body-l-size, etc.
Don't
Set font sizes in px directly in component CSS. Use the tokens. This is how the system stays coherent across breakpoints.
Do
Use weight 700 for eyebrow labels, headings, and strong emphasis. Weight 400 for running body text. Weight 300 for captions only.
Don't
Mix Fraunces and Bai Jamjuree in the same heading. One head = one typeface. Fraunces for the largest, Bai for sub-headers at h2 and below.
Upgrade Path

GT Sectra Display — when the budget allows.

Fraunces is the approved substitute until the commercial license is acquired. Fraunces and GT Sectra are not interchangeable in feel — Sectra has a scalpel angularity that Fraunces doesn't reproduce. When licensed, update the --font-display CSS variable. Nothing else in the codebase changes.

Now — Fraunces (free)
  • SIL OFL — free commercial
  • Google Fonts — CDN available
  • Variable font — weight + optical size
  • Warm, humanist, personality-forward
  • Approved for all NICHE usage
Target — GT Sectra Display (licensed)
  • Commercial license — Grilli Type
  • Designed for Reportagen magazine
  • Calligraphic warmth × scalpel angularity
  • 30 styles across Display / Fine / Text
  • Count seats accurately before licensing

License GT Sectra Display from grillitype.com. Desktop license: CHF 119–399 depending on seats. Web license: based on monthly pageviews. Count seats before purchasing. Once licensed: update --font-display in tokens.css, update the @import in the Google Fonts link tag, verify rendering across OS (macOS, Windows, Android, iOS).
