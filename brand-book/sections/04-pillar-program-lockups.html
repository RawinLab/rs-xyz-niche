Section 04

Pillar & Program Lock-Ups

How the 3 pillars and 12 programs sit within the brand architecture.

Overview

Brand architecture hierarchy

NICHE (institution) → Pillar (Creativity / Humanity / Entrepreneurship) → Program (e.g. AI x Content Engine Studio). Three levels. No program operates without its pillar context.

Lock-up system principle

Lock-up system principle

A lock-up is NICHE wordmark + pillar label + program name, in a fixed relationship. The system means any of the 12 programs can be identified consistently without each needing its own bespoke design.

Pending Content
