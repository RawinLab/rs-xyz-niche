Section 03

Logo System

Three logos. One institution. Each with a defined role, scope, and permission set. The system answer — not the compromise answer.

Primary Wordmark

OPT 02 — Institutional backbone.

Bold rounded humanist sans with a tilted N. The tilt is a genuine mnemonic — it makes NICHE recognizable at a glance and creates a natural motion behavior. The asymmetric C differentiates it from any font in current circulation. This wordmark represents NICHE in all institutional contexts.

Both light and dark colorways are approved. Match the background, never approximate.

Wordmark form All-caps custom drawn wordmark — not a typeface
Distinctive features Tilted N · Asymmetric open C · Rounded stroke terminals
Register Confident · editorial · magazine-cover · approachable

On light (approved) #252026 on #f2f1f0

Submark

N. — Protect the period.

The house-shaped N with a period is the canonical submark of NICHE. The period is doing genuine concept work: a niche is a carved recess, and the period punctuates that claim. It also reads as a confident declarative sentence-ender — NICHE. as a statement. It will outlive the wordmark at favicon, app icon, IG avatar, embroidery, and embossing scale.

Form House-shaped N (roof peaks) + square period · custom drawn
Approved uses Favicon · app icon · IG avatar · embroidery · embossing · foil · corner mark on collateral
Minimum print 12mm height. Below this the period detaches visually.
Minimum digital 32×32px. At 16px favicon size, use solid N without period.
Do not Remove the period. The period is the concept — not punctuation.

Clear Space

The cap-height of the E. Nothing intrudes.

Maintain a minimum clear zone equal to the cap-height of the letter E in the wordmark on all four sides. This is the exclusion zone — no other logos, text, images, or graphic elements may enter it.

1× cap-height 1× cap-height 1× 1×
Unit 1× the cap-height of E in the wordmark at its rendered size
Example at A4 Wordmark at 40mm wide → E cap ≈ 8mm → 8mm exclusion all sides
Example at web (200px wide) E cap ≈ 34px → 34px padding all sides
Applies to All wordmark + submark uses in all orientations

Do / Don't

Every violation listed here has happened. Don't let it happen again.

These are not preferences — they are rules. Each violation destroys a property that makes the mark distinctive.

Don't — Stretch Never scale disproportionately. Lock aspect ratio always.
Don't — Rotate Never rotate the wordmark. The tilted N is intentional — adding rotation conflicts with it.
Don't — Recolor Only use the three approved colorways. No custom colors, no gradients.
Don't — Low Contrast Never place the wordmark on a background that fails 3:1 contrast minimum.
Don't — Drop Shadow The mark is flat. No shadows, glows, emboss effects, or dimensional treatments.
Don't — Remove Period
N
The "N." submark always includes the period. The period is the concept, not punctuation.
Do
Use the supplied logo files. Never recreate the wordmark in software. Always embed the vector file — never screenshot or rescale a raster export.
Don't
Set "NICHE" in any typeface and call it the logo. Even the closest available geometric sans will not match the custom drawn letterforms.
Do
Use the N. submark as a standalone mark at small sizes. It is system-canonical and holds identity where the wordmark cannot.
Don't
Use OPT 03 (thin art-deco) for any institutional communication — namecards, signage, diplomas, website header. It is scoped to Journal only.
File Formats

Right file for the right job.

The logo package ships in four formats. Use the correct one for each context. Never use a JPG where SVG is possible.

.svg
Web / Screen
Use for all web, app, and digital-production contexts. Scales infinitely.
.pdf
Print
For print vendors. Vector PDF, all fonts outlined, CMYK color profile.
.png
Raster fallback
2000px wide, transparent BG. For contexts that don't support SVG (Notion, Canva, Office).
.eps
Legacy print
For older print and signage workflows requiring EPS. Spot-color version available.
