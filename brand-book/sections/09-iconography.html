Section 09

Iconography & Graphic Language

How NICHE builds visual language beyond the logo.

Overview

No decorative icon system

NICHE does not use a conventional UI icon set with personality. UI icons (navigation, UI controls) should use a neutral, geometric system — not a branded custom set. The visual personality lives in the logo, type, and photography — not icons.

Collage technique

Collage technique

The composite layout system (page 18) shows NICHE's graphic language: overlapping block-color tiles at irregular proportions, anchored by the near-black #252026 base. This is the poster and campaign composition principle.

Poster principles

Poster principles

Every NICHE poster anchors on a dominant text element (not an image). The image supports the text, not the reverse. Fraunces at maximum scale. Negative space is generous. Nothing is centered unless the centering is the design.

Pending Content

Brand owner input required.

Icon library selection: Select and document a third-party icon set that matches NICHE's aesthetic: geometric, minimal, no rounded-corner style. Phosphor Icons or Material Symbols Outlined (weight 200) are starting candidates.

Illustration policy: Does NICHE use illustration? If so, what style? Hand-drawn, geometric, photographic collage? Policy required before vendor briefs.
