Color Palette

Five colors. The tension between near-black, vivid red, and muted green is the identity. A sixth color dilutes it. Don't add one.

WebAIM 2024: 83.6% of home pages have detectable WCAG contrast failures. NICHE's red (#fe1d25) and warm gray (#cccecd) both fail body-text contrast on light backgrounds. This is documented and enforced — not ignored. Design systems that document known failures prevent them. Systems that don't document failures repeat them.

Hard Rules

Three prohibitions. No exceptions.

Red — display only

#fe1d25 is for display text only — ≥18pt regular or ≥14pt bold. Never use as a hyperlink color on light backgrounds. Never use for body text at any size.

Warm gray — decorative only

#cccecd is for dividers, disabled states, and decorative backgrounds. Never use as text color on any background. Never use as a background for text elements.

Green — dark background only for body

#779152 as body text is only approved on #252026. On #f2f1f0, use for large display only. For body text on light — use #4a5c33 (accessible shade).

Do
Use red for section numbers, eyebrow labels, pull-quote accents, CTAs. Let it punctuate — never let it flood.
Don't
Use red as a hyperlink color, as body copy color, or as a paragraph background on any white/off-white surface.
Do
Use #4a5c33 (accessible green shade) for any body copy that must appear on #f2f1f0 in green. It passes 7.2:1 — more than enough headroom.
Don't
Use the warm gray (#cccecd) as placeholder text or secondary text. It reads as broken UI. Use rgba(37,32,38,0.5) for muted text instead.

Semantic Tokens

Don't reference raw hex in code.

Dark Mode Token Alternates

The brand was authored light-first. Dark mode is explicit.

#252026 does not auto-translate to system dark mode without contrast loss. WCAG requires 4.5:1 body text regardless of mode. These alternates must be defined in Figma Variables and CSS custom properties before any dark-mode web build launches.

Dark-mode token values require a final decision from the brand owner: Should the dark background be #252026 (the brand near-black) or pure #111? Should the dark surface text be #f2f1f0 or #ffffff? All token alternates below are provisional pending that decision. Confirm, then author the @media (prefers-color-scheme: dark) block in tokens.css.

Token Light value Dark mode alternate (provisional)

Print Color Specs

CMYK and Pantone references for print vendors.

Pantone references require confirmation against physical Pantone swatchbooks — the values below are starting approximations only. Confirm with your print partner before the first production run. For spot-color jobs, the near-black should ideally use a custom mix (not process black) to preserve the slight warm-purple undertone of #252026. The red should be confirmed against Pantone 485 C or 2347 C on coated stock.
